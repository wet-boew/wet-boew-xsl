Ce plugiciel créé un élément qui peut être utilisé afin d'agrandir ou de réduire tous les éléments de details sur une page.

Exemples

Afficher le code 
<div class="wet-boew-expandcollapseall toggle toggle-open toggle-close accent-first print-open"></div>
Afficher le code 
<div class="wet-boew-expandcollapseall toggle-open toggle-close"></div>
Afficher le code 
<div class="wet-boew-expandcollapseall"></div>

Les classes CSS peuvent-être utilisées pour configurer le comportement de l'élément de basculement. Celles-ci peuvent également être configurées à l'intérieur du fichier de remplacement settings.js :

Options de configuration CSS
Class Détails
wet-boew-expandcollapseall Créer un élément de basculement. Sans aucune autre option, ceci créera un seul bouton à bascule.
accent-first Accentuer le premier bouton.
print-open Agrandir automatiquement les details lors de l'impression.
toggle Ajouter un bouton qui sert à agrandir et à réduire les éléments de details.
toggle-close Ajouter un bouton qui sert seulement à réduire les éléments de details.
toggle-open Ajouter un bouton qui sert seulement à agrandir les éléments de details.
