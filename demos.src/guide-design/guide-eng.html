Any Web site using the WET templates should use and adhere to this guide.

This guide features the intended use of the CSS and scripting features. By adhering to the correct use of each style, a site is well positioned to upgrade to future template versions. Sites that do not adhere to the design guide may find that their templates, CSS and JavaScripts are out-of-step with subsequent releases.
